# Ledgerly

Ledgerly is a full-stack web application for managing personal finances, built with a Node.js/Express backend and a React/Vite frontend. 
It allows users to register, log in, manage wallets, track transactions and learn more about financial literacy.

## Features
- User authentication (register, login)
- Wallet management
- Transaction tracking
- Financial dashboard
- Educational resources
